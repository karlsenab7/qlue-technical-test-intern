## Setup
```
npm install fs
npm install express
npm install axios
```
___
## Run server using command :

```
node index.js
```
___
## Get Raw Data
### Get the raw json file through this link
```
http://localhost:3000/getRawData
```
___
## Get Parsed Data
### Get the parsed json file through this link
```
http://localhost:3000/getParsedData
```
